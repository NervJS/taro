# Taro Components for React Native

Alo, alo! Bilibilibilibibili~

## Example

```bash
# 启动安卓模拟器
emulator @YOUR_AVD_NAME

# 安装依赖
yarn

# 编译
npm run dev

# 进入例子目录安装依赖
cd TCRNExample
yarn

# 启动 Android
react-native run-android
# 启动 iOS
react-native run-ios
```
