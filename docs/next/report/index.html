接入统计平台

#接入百度统计平台

#微信小程序接入百度统计

1、在百度统计平台 的 「小程序SDK」 里下载「微信小程序统计SDK」，把解压缩得到的 mtj-wx-sdk.js 和 mtj-wx-sdk.config.js 拷贝到小程序项目的 src/utils 文件夹中。

2、编辑 mtj-wx-sdk.config.js，填写 appKey 字段:

appKey: '您小程序的AppKey',

3、编辑小程序项目的 src/app.tsx，在文件最顶部加入：

import './utils/mtj-wx-sdk';

4、登录微信公众平台，进入小程序的 设置 -> 开发设置 -> 服务器域名，把https://hmma.baidu.com加入 request合法域名。

详细文档见https://mtj.baidu.com/static/userguide/book/chapter0/wechat.html#34-taro
