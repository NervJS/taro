## 方法

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| accuracy | `number` | iBeacon 设备的距离 |
| major | `string` | iBeacon 设备的主 id |
| minor | `string` | iBeacon 设备的次 id |
| proximity | `number` | 表示设备距离的枚举值 |
| rssi | `number` | 表示设备的信号强度 |
| uuid | `string` | iBeacon 设备广播的 uuid |
