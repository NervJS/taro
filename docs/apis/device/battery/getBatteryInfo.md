获取设备电量。同步 API Taro.getBatteryInfoSync 在 iOS 上不可用。

> [参考文档](https://developers.weixin.qq.com/miniprogram/dev/api/device/battery/wx.getBatteryInfo.html)

## 类型

```tsx
(option?: Option) => void
```
