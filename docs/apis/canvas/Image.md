图片对象

> [参考文档](https://developers.weixin.qq.com/miniprogram/dev/api/canvas/Image.html)

## 方法

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| height | `number` | 图片的真实高度 |
| onerror | `(...args: any[]) => any` | 图片加载发生错误后触发的回调函数 |
| onload | `(...args: any[]) => any` | 图片加载完成后触发的回调函数 |
| src | `string` | 图片的 URL |
| width | `number` | 图片的真实宽度 |
