使用方式同 [`wx.navigateTo`](https://developers.weixin.qq.com/miniprogram/dev/api/wx.navigateTo.html)，支持 `Promise` 化使用。

## 示例代码

```jsx
import Taro from '@tarojs/taro'

Taro.navigateTo(params).then(...)
```
