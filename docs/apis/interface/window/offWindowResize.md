取消监听窗口尺寸变化事件。

## 参数

### callback(res)

窗口尺寸变化事件的回调函数。

#### 参数

##### object res

| Name | Type | Description |
| --- | --- | --- |
| windowWidth | <code>number</code> | 变化后的窗口宽度，单位 px |
| windowHeight | <code>number</code> | 变化后的窗口高度，单位 px |
