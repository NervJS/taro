创建激励视频广告组件。

> [参考文档](https://developers.weixin.qq.com/miniprogram/dev/api/ad/wx.createRewardedVideoAd.html)

## 类型

```tsx
(option: Option) => RewardedVideoAd
```

## 参数

### Option

| 参数 | 类型 | 默认值 | 必填 | 说明 |
| --- | --- | :---: | :---: | --- |
| adUnitId | `string` |  | 是 | 广告单元 id |
| multiton | `boolean` | `false` | 否 | 是否启用多例模式 |
