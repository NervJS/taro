在input、textarea等focus拉起键盘之后，手动调用此接口收起键盘

> [参考文档](https://developers.weixin.qq.com/minigame/dev/api/ui/keyboard/wx.hideKeyboard.html)

## 类型

```tsx
(option?: Option) => void
```
