#组件属性

NameTypeDescription
frontColor
string
前景颜色值，包括按钮、标题、状态栏的颜色，仅支持 #ffffff 和 #000000
backgroundColor
string
背景颜色值，有效值为十六进制颜色
animation
Object
动画效果
[success]
function
接口调用成功的回调函数
[fail]
function
接口调用失败的回调函数
[complete]
function
接口调用结束的回调函数（调用成功、失败都会执行）
