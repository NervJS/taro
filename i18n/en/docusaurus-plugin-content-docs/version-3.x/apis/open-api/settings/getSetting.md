Obtains the user's current settings. **Only the [permissions](https://developers.weixin.qq.com/miniprogram/en/dev/framework/open-ability/authorize.html) that have been requested by the Mini Program from the user are returned.**

> [Reference](https://developers.weixin.qq.com/miniprogram/en/dev/api/open-api/setting/wx.getSetting.html)

## Type

```tsx
(option?: Option) => Promise<SuccessCallbackResult>
```
