---
title: Taro.hideHomeButton(option)
sidebar_label: hideHomeButton
---

Hide the button to return to the home page. Since WeChat 7.0.7, when the user opens the root page of an mini program that is not the home page, the "Back to Home" button will be displayed by default, developers can call hideHomeButton in the page onShow to hide it.

> [Reference](https://developers.weixin.qq.com/miniprogram/dev/api/ui/navigation-bar/wx.hideHomeButton.html)
