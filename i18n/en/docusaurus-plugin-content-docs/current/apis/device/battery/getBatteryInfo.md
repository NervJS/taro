Obtains the device's battery level. The synchronous API `Taro.getBatteryInfoSync` is not available on iOS.

> [Reference](https://developers.weixin.qq.com/miniprogram/en/dev/api/device/battery/wx.getBatteryInfo.html)

## Type

```tsx
(option?: Option) => void
```
