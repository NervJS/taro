## 示例代码

```tsx
Taro.getNetworkType({
  success: function (res)) {
    // 返回网络类型, 有效值：
    // wifi/2g/3g/4g/unknown(Android下不常见的网络类型)/none(无网络)
    var networkType = res.networkType
  }
})
```
